# Personal Task Tracker

## Description

**Personal Task Tracker** is a lightweight React-based web app that allows users to log in with a username and manage their personal tasks. The app features task creation, editing, filtering, completion tracking, and productivity stats â€” all stored in the browser using `localStorage` per user.

---

## Features

- Simple username-based login (no password)
- Add tasks with title & description
- Mark tasks as completed or pending
- Edit existing tasks inline
- Delete tasks with confirmation
- Search tasks by title
- Filter tasks: All | Completed | Pending
- Dashboard with:
  - Task count & stats
  - Productivity metrics
  - Current day display
  - Motivational quote
- Logout (clears current user)
